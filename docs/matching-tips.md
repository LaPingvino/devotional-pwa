## Cross-language parallel books

- **diu/kj/hz**: Same Namibian prayer book structure (useful for cross-validation)
- **ny (Chichewa)**: Good morning prayer reference (BB00335→BH00009SHE sequence)
- **lo (Lao)**: Good victory/teaching reference; has PIN codes for unmatched
